# Исполнитель конечных автоматов Мура и Мили

Создание полноценного исполнителя для упрощения создания конечных автоматов Мура и Мили, получающих на вход двоичное число.

**Статус проекта:** Активная разработка. API может меняться!

## Текущая структура

### Новый код (активная разработка)

- **`src/state_machines/`** — ядро проекта. Здесь разрабатывается новый API для конечных автоматов:
  - [`mealy_machine.py`](src/state_machines/mealy_machine.py) — основной класс `MealyMachine`
  - [`mealy_state.py`](src/state_machines/mealy_state.py) — состояния и переходы
  - [`mealy_step.py`](src/state_machines/mealy_step.py) — результаты выполнения

- **`src/gui/`** — графический редактор для визуального создания автоматов:
  - [`main.py`](src/gui/main.py) — точка входа в GUI-редактор
  - [`colors_tuples.py`](src/gui/colors_tuples.py) — цветовая схема

- **`tests/`** — тесты для нового API:
  - [`mealy_machine/`](tests/mealy_machine/) — тесты для автоматов Мили
  - [`utils/`](tests/utils/) — вспомогательные функции для тестирования

### Legacy (устаревший код, поддержка не гарантируется)

- **`_legacy/`** — старые версии, оставлены для совместимости:
  - [`examples/`](_legacy/examples/) — примеры на старый API
  - [`state_machines/`](_legacy/state_machines/) — старая реализация автоматов (Mealy, Moore)
  - [`tests/`](_legacy/tests/) — тесты для устаревшего кода
